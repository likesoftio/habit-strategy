# habit-strategy

Презентация «17 ритуалов за 16 недель» — `https://likesoftio.github.io/habit-strategy/`.

Стратегия внедрения 17 привычек за 4 фазы по ~4 недели каждая. Habit stacking, keystone-привычки, implementation intentions, identity-based change.

## Стек

Чистый HTML + CSS + vanilla JS. Без билда, без зависимостей. Шрифты подгружаются с Google Fonts. Деплой — GitHub Pages с корня `main`.

## Управление

- `←` / `→` / `Space` — переключение слайдов
- `Home` / `End` — первый / последний
- Свайп на тач-устройствах
- `↓ PDF` — экспорт в PDF через `window.print()`
